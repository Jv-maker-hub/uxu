Cómo usar esta guía

Una guía,
no un folleto

Esta web es una herramienta de trabajo, no un documento para leer una vez y archivar. Está hecha para que cualquier persona que cree contenido, diseñe piezas o planifique anuncios para UNO × UNO tenga una referencia clara de qué decir, cómo decirlo y cómo no desviarse del tono de la marca.

El orden sugerido de lectura es: Buyer Personas → Competidores → Manual de Marca → Tipografías → Briefs de Ads. No hace falta leer todo de corrido. Cada documento funciona solo.
