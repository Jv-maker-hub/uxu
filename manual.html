Identidad Visual

Logotipo

El logotipo se desarrolla a partir de Trade Gothic Condensed Bold, utilizada como punto de partida para definir proporciones, ritmo y estructura. A partir de esta base tipográfica se realizaron ajustes para construir un logotipo propio, coherente y consistente.

Su diseño prioriza la claridad, el equilibrio y la repetición, reforzando el concepto de sistema y uniformidad. La "×" funciona como elemento central: representa el cruce, la repetición y la consistencia en el uso diario. No es un recurso decorativo. Es una pieza funcional que articula la marca.

Color principal

El gris carbón (#292A2E) es el color principal de la marca y debe utilizarse prioritariamente en el logotipo y elementos de identificación.

Se optó por un gris carbón —y no negro puro— para lograr una presencia más equilibrada y atemporal. Convive mejor con blancos y neutros, permite jerarquías más sutiles, y no cansa en repetición.

Negro pleno solo en casos técnicos (por ejemplo, impresión en una tinta muy limitada).

Packaging

El packaging refleja el sistema de la marca: claridad, simplicidad y funcionalidad. Se resuelve con una estructura limpia, donde el producto es protagonista. Se utilizan pocos elementos gráficos, organizados con aire y jerarquía clara.

El logotipo se aplica de forma directa y la comunicación se reduce a una única frase.

Caja

  • Color cream. Ventana para visualizar el producto.
  • Logotipo positivo + única frase de comunicación.
  • Sin mensajes promocionales ni urgencias.

Bolsa de envío

  • Bolsa tipo courier en color Gris Carbón (#292A2E).
  • Impresa a una tinta. Logotipo + frase breve.
  • Sin recursos comerciales ni mensajes forzados.

Sticker de talle

  • Función informativa dentro del packaging.
  • Prioriza claridad y lectura inmediata sin abrir el producto.
  • Información organizada en niveles: material → origen → talle.
